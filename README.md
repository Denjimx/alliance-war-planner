# Alliance War Planner by DenjiMX

Herramienta responsive para planear defensas de Alliance War en Marvel Contest of Champions.

## Versión actual

**v4.0.0 responsive**: un solo `index.html` para computadora, tablet y celular.

- En computadora, arrastra campeones al mapa o selecciónalos y haz clic.
- En celular, selecciona un campeón y toca el mapa para colocarlo.
- Zoom con rueda del mouse, botones o gesto de pellizco.
- Guardado local, importación/exportación de proyectos y exportación PNG.

## Publicación

GitHub Pages publica siempre la rama configurada del repositorio. Al hacer `push` a `main`, la misma URL se actualiza automáticamente:

`https://denjimx.github.io/alliance-war-planner/`

El antiguo enlace `mobile.html` redirige a la versión unificada.

## Archivos de respaldo

- `index-desktop-legacy.html`: versión anterior de escritorio.
- `mobile-legacy.html`: respaldo de la base móvil usada para la unificación.
